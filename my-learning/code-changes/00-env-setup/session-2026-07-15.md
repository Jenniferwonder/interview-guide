# 会话代码改动清单（2026-07-15）

> 本会话在本地排障与体验优化中落地的代码/配置变更（不含 git commit）。  
> 踩坑说明见 [README.md](README.md) 第 4 节与对应 notes。

## 改动一览

| 改动 | 文件 | 说明 |
|------|------|------|
| TTS 开场白启动预热开关 | `VoiceInterviewProperties.java`、`VoiceInterviewWebSocketHandler.java`、`application.yml` | 新增 `warmupOpeningAudioEnabled` / `app.voice-interview.warmup-opening-audio-enabled`，默认 `false`；开发启动不再打云端 TTS |
| 关闭 Spring Data Redis 仓库扫描 | `application.yml` | `spring.data.redis.repositories.enabled: false`，消除启动 INFO 噪声 |
| 面试列表投影接线 | `InterviewController.java`、`InterviewPersistenceService.java`（import） | `listSessions` 改调 `findAllListItems()`，不再 `findAll()` 加载 TEXT |
| 列表投影查询（已有/补齐） | `InterviewSessionRepository.findAllListItems`、`SessionListItemDTO` | JPQL 只取列表列 |
| 索引列名统一 | `InterviewSessionEntity.java` | `skillId,createdAt` → `skill_id,created_at`；保留 `created_at` 索引 |
| ddl-auto 保持 update | `application.yml` | 避免 `create` 重启删表；注释提醒首次建表后改回 `update` |

## 配置摘要

```yaml
spring:
  jpa:
    hibernate:
      ddl-auto: update
  data:
    redis:
      repositories:
        enabled: false

app:
  voice-interview:
    warmup-opening-audio-enabled: ${APP_VOICE_INTERVIEW_WARMUP_OPENING_AUDIO_ENABLED:false}
```
